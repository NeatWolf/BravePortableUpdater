# Code Of Conduct

This project is small and practical. Keep discussion focused on making the
updater safer, clearer, and easier to verify.

## Expected Behavior

- Be direct, specific, and respectful.
- Ground claims in commands, logs, screenshots, file paths, or reproduction
  steps when possible.
- Keep reports scoped to this updater, Portapps-style Brave Portable behavior,
  and the repository documentation.
- Respect privacy. Do not post browser profile contents, cookies, credentials,
  wallet data, or other private user files.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language.
- Publishing private profile data or security-sensitive details.
- Submitting unrelated binaries, generated bundles, or browser profile data.
- Pressuring maintainers to weaken update safety checks.

## Enforcement

The repository owner may edit, hide, close, or remove contributions that violate
these expectations. Security-sensitive issues should be reported privately using
[SECURITY.md](SECURITY.md).
